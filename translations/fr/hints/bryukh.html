Tout d'abord, nous devrions trouver des mots. Nous connaissons déjà str.split().

"asd,dfg,asd".split(",") == ["asd", "dfg", "asd"]
J'ai besoin d'aide pour poursuivre la mission

Pour cette mission, nous pouvons utiliser les outils intégrés et le type de données utile sets.

a = set([1, 2, 3])
b = set([1, 2])

a.intersection(b) == {1, 2} # or just a & b
Je suis à mi-chemin. J'ai besoin d'aide!

N'oubliez pas de trier les données des résultats.

result = ["one", "two", "three"]
sorted(result) == ["one", "three", "two"]
result = sorted(result) # or just result.sort()
Je suis bloqué. J'ai besoin d'un petit conseil.

str.join() est important ici.

",".join(["a", "b", "c"]) == "a,b,c"
Alors, à quoi devrait ressembler ma solution finale?

La voici:

def checkio(first: str, second: str) -> str:

    first = set(first.split(","))
    second = set(second.split(","))

    return ",".join(sorted(first & second))

Il existe de nombreuses autres solutions que vous pouvez voir lorsque votre code passe tous les tests. Par exemple, cette solution fait la même chose, mais en une seule ligne:

checkio=lambda f,s,c=lambda x:set(x.split(",")):",".join(sorted(c(f)&c(s)))

Cette solution fait la même chose, mais en une seule ligne en utilisant deux lambdas. L'autre solution est appelée "oneliner":

def checkio(first: str, second: str) -> str:

return ",".join(sorted(set(first.split(",")).\
                       intersection(set(second.split(",")))))
