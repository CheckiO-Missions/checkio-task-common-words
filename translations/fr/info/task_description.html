Poursuivons l'examen des mots. On vous donne deux chaînes de mots séparés par des virgules. Essayez de trouver ce qui est commun à ces chaînes. Les mots d'une même chaîne ne se répètent pas.

Votre fonction doit trouver tous les mots qui apparaissent dans les deux chaînes. Le résultat doit être représenté sous la forme d'une chaîne de mots séparés par des virgules dans l'ordre alphabétique.

Entrée: Deux arguments sous forme de chaînes de caractères (str{% if is_js %}ing{% endif %}).

Sortie: Les mots communs sous forme de chaîne (str{% if is_js %}ing{% endif %}).

{{init_code_tmpl}}

Comment on l'utilise: Vous apprendrez ici à travailler avec des chaînes et des ensembles. Ces connaissances peuvent être utiles pour l'analyse linguistique.

Conditions préalables:

  • Chaque chaîne ne contient pas plus de 10 mots;
  • Tous les mots sont séparés par des virgules;
  • Tous les mots sont composés de lettres latines minuscules.
