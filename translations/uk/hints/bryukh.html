Тобі буде корисним метод рядків str.join(). 
",".join(["a", "b", "c"]) == "a,b,c"
То як має виглядати моє остаточне рішення?
Ось воно: 

def checkio(first: str, second: str) -> str:

    first = set(first.split(","))
    second = set(second.split(","))

    return ",".join(sorted(first & second))
Є багато інших розв'язків цієї місії, які ти зможеш побачити коли твій код успішно пройде всі тести. Наприклад: 

checkio=lambda f,s,c=lambda x:set(x.split(",")):",".join(sorted(c(f)&c(s)))
Цей розв'язок робить те ж саме, але в один рядок із використанням двох lambda виразів. І це інший так званий ванлайнер: 

def checkio(first: str, second: str) -> str:
return ",".join(sorted(set(first.split(",")).\
                       intersection(set(second.split(",")))))
