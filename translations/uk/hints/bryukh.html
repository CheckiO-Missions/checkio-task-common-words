str.join() тут дуже важлива.

Отже, як має виглядати моє остаточне рішення?

Ось воно:

def checkio(first: str, second: str) -> str:

    first = set(first.split(","))
    second = set(second.split(","))

    return ",".join(sorted(first & second))

Є багато інших рішень, які ви зможете побачити, коли ваш код пройде всі тести. Наприклад:

checkio=lambda f,s,c=lambda x:set(x.split(",")):",".join(sorted(c(f)&c(s)))

Це рішення робить те ж саме, але в одному рядку, використовуючи дві лямбди. Інший так званий oneliner:

def checkio(first: str, second: str) -> str:

return ",".join(sorted(set(first.split(",")).\
                       intersection(set(second.split(",")))))
