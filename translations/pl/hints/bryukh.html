Nie mam pojęcia, jak zacząć rozwiązywać tę misję

Na początku powinniśmy zdobyć słowa. Wiemy już o str.split().

"asd,dfg,asd".split(",") == ["asd", "dfg", "asd"]
Potrzebuję pomocy, aby kontynuować misję

W tej misji możemy użyć wbudowanych narzędzi i przydatnego typu danych sets.

a = set([1, 2, 3])
b = set([1, 2])

a.intersection(b) == {1, 2} # or just a & b
Jestem w połowie. Proszę o pomoc!

Proszę nie zapomnieć posortować danych wynikowych.

result = ["one", "two", "three"]
sorted(result) == ["one", "three", "two"]
result = sorted(result) # or just result.sort()
Utknąłem. Proszę o małą podpowiedź.

str.join() jest tutaj ważne.

",".join(["a", "b", "c"]) == "a,b,c"
Jak powinno wyglądać moje ostateczne rozwiązanie?

Proszę bardzo:

def checkio(first: str, second: str) -> str:

    first = set(first.split(","))
    second = set(second.split(","))

    return ",".join(sorted(first & second))

Istnieje wiele innych rozwiązań, które można zobaczyć, gdy kod przejdzie wszystkie testy. Na przykład:

checkio=lambda f,s,c=lambda x:set(x.split(",")):",".join(sorted(c(f)&c(s)))

To rozwiązanie robi to samo, ale w jednej linii za pomocą dwóch lambd. Inne rozwiązanie to tak zwany oneliner:

def checkio(first: str, second: str) -> str:

return ",".join(sorted(set(first.split(",")).\
                       intersection(set(second.split(",")))))
Dzięki! Teraz wiem i rozumiem więcej! Wszystko działa dobrze!
Miło mi to słyszeć!
