Kontynuujmy analizę słów. Otrzymują Państwo dwa ciągi ze słowami oddzielonymi przecinkami. Proszę spróbować znaleźć to, co jest wspólne między tymi ciągami. Słowa w tym samym ciągu nie powtarzają się.

Funkcja powinna znaleźć wszystkie słowa występujące w obu ciągach. Wynik musi być przedstawiony jako ciąg słów oddzielonych przecinkami w kolejności alfabetycznej.

Dane wejściowe: Dwa argumenty jako ciągi znaków (str{% if is_js %}ing{% endif %}).

Dane wyjściowe: Wspólne słowa jako ciąg znaków (str{% if is_js %}ing{% endif %}).

Jak jest używany: Tutaj dowiedzą się Państwo, jak pracować z ciągami i zbiorami. Wiedza ta może być przydatna w analizie językowej.

Warunki wstępne:

  • Każdy ciąg zawiera nie więcej niż 10 słów;
